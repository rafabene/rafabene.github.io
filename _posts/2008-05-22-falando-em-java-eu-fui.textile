title: Falando em Java! Eu fui!
---

<p>Divertido!!! Esta com certeza é a palavra que eu usuaria para melhor definir o evento realizado neste domingo. <img style="max-width: 800px; display: block;" src="/assets/images/falandoemjava.jpg" />A primeira palestra foi realizada por Guilherme Silveira, falando sobre os 7 hábitos dos arquitetos altamente eficazes. Esta palestra demonstrou como seria o <i>"tom"</i> de todo o evento, apresentando fatos técnicos de uma maneira <u>divertida</u>. Sem dúvida alguma assino embaixo dos 7 hábitos citados. Em seguida, Sérgio Lopes, "desmistificou" o DDD (Domain Driven Design) através de uma <u>divertida</u> dramaturgia onde contracenava com um "cachorro falante". Finalmente foi a vez de <a target="_blank" href="http://blog.emmanuelbernard.com/">Emmanuel Bernard</a> falar da <a target="_blank" href="http://www.jcp.org/en/jsr/detail?id=317">JSR 317 - JPA 2.0</a> onde me chamou bastante a atenção algumas novidades:<ul><li>Automatic Orphan detection</li><li>Introdução da API de Criteria</li><li>setLockMode para o objeto <a target="_blank" href="http://java.sun.com/javaee/5/docs/api/javax/persistence/Query.html">Query</a> e também para a anotação @<a target="_blank" href="http://java.sun.com/javaee/5/docs/api/javax/persistence/NamedQuery.html">NamedQuery</a> - Isto com certeza deve acabar com as "brigas" de muitos DBAs.</li><li>Algumas propriedades do persistence.xml (driver, url, usuário e senha) também serão padronizadas.</li><li>entre várias outras</li></ul>Também foi falado nesta palestra sobre a <a target="_blank" href="http://www.jcp.org/en/jsr/detail?id=303">JSR 303</a> - Bean Validation.
  
Após o almoço, <a target="_blank" href="http://blog.emmanuelbernard.com/">Emmanuel Bernard</a> voltou a falar do <a target="_blank" href="http://www.hibernate.org/410.html">Hibernate Search</a>, o qual eu nunca havia usado antes, mas que realmente traz o poder (com grande facilidade) do <a target="_blank" href="http://lucene.apache.org/">Lucene</a> para a Persistência e sem dependência de mecanismos dos Bancos de Dados para <a target="_blank" href="http://en.wikipedia.org/wiki/Full_text_search">Full-Text-Search</a>. <a target="_blank" href="http://fabiokung.com/about/minicurriculo-pt_br/">Fábio Kung</a> apresentou o JRuby on Rails e a novidade de que a versão 3.0 do <a target="_blank" href="http://www.guj.com.br/">GUJ</a> será feita com esta tecnologia. É inegável o papel que as linguagens dinâmicas estão tendo atualmente, mas ainda não consegui sair evangelizado <img src="http://www.jroller.com/images/smileys/smile.gif" class="smiley" alt=":)" title=":)" /> - Apesar da minha (pouca e recente) vivência com o <a target="_blank" href="http://www.rubyonrails.org/">Rails</a>, ainda acredito no <a target="_blank" href="http://www.seamframework.org/">Jboss Seam</a> como framework de produtividade e escalabilidade para Web 2.0. Fábio Silveira então apresentou rapidamente os projetos Open Source da <a target="_blank" href="http://www.caelum.com.br/caelum/index.jsp">Caelum</a> e então Guilherme Moreira e Thadeu Russo apresentaram de maneira <u>divertida</u> as possibilidades de uso do Ajax para construção de interfaces Web mais interativas.Para fechar com chave de ouro, <a target="_blank" href="http://amagno.blogspot.com/">Alexandre Magno</a> e Edmilson Miyasaki fizeram uma <u>divertida</u> palestra sobre Scrum, citando a realidade do desenvolvimento para finalmente mostrar como o Scrum tem auxiliado a mudar o <a target="_blank" href="http://www.projectsmart.co.uk/docs/chaos-report.pdf">Chaos Report</a>. (Uma excelente <b>introdução</b> ao Scrum pode ser encontrada no livro <a target="_blank" href="http://www.infoq.com/minibooks/scrum-xp-from-the-trenches">Scrum and XP from the Trenches </a>). É válido destacar que uma boa implantação do Scrum somente é feita quando toda a equipe interna foi treinada.No final do evento, aconteceu o sorteio do Nitendo Wii através de uma rotina escrita em Ruby. Infelizmente (no meu caso) a sorte passou perto, bem perto mesmo. O cara (desculpe por não saber o nome) que estava sentado ao meu lado ganhou. :/Entre um intervalo e outro, além da grande quantidade de iPhones e Macbooks, tive a oportunidade de conhecer alguns pessoalmente (comunidade virtual é assim mesmo), fazer novas amizades e rever alguns amigos. 

A todas estas pessoas, um grande abraço!<img style="max-width: 800px; display: block;" src="/assets/images/edgaralexandre.jpg" /></p>
